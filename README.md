## Para Backend

# Configurar entorno
* python -m venv chatbot-venv
* .\chatbot-venv\Scripts\activate

# Instalar requerimientos
* pip install -r requirements.txt

# Correr el servidor
* uvicorn main:app --reload --port 8000

## Para Frontend

# Instaslar requerimientos
* npm install

# Correr el servidor (modo dev)
* npm run dev
